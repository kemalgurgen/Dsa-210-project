# DSA 210 Project — May 5 Machine Learning Milestone

This folder contains the May 5 milestone for the cryptocurrency price prediction project.

## Goal

Apply machine learning methods on the processed dataset created in the April 14 milestone.

## Dataset

The notebook uses `final_dataset.csv`, which combines BTC OHLCV data with USDT market indicator data.

I included `final_dataset.csv` in the repository so the notebook can be tested and reproduced if needed.

## Models Used

- Logistic Regression
- Decision Tree
- Random Forest

## Outputs

- `may5_ml_dataset.csv`
- `may5_model_comparison.csv`

## How to Run

1. Upload `crypto_analysis_may5_milestone.ipynb` to Google Colab.
2. Upload `final_dataset.csv`.
3. Click Runtime → Restart runtime.
4. Click Runtime → Run all.

AI was used for technical assistance in structuring and debugging the machine learning notebook. All analysis decisions and final interpretations were reviewed by the student.
